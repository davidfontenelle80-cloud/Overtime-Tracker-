# Overtime Tracker

UConn Facilities overtime equalization tracker. Tracks OT offers (offered, accepted, declined), hours worked by type (OT, vacation, sick, FMLA, comp, block, period), and equalization balance across crew members. Dark/light theme, responsive.

**Zoom:** locked (`user-scalable=no`, `maximum-scale=1.0`) per owner decision — accidental pinches destabilize the app in field use; all text-entry inputs stay ≥16px, so readability and iOS auto-zoom prevention hold regardless (KHub UX-STANDARDS §2).
