# Overtime Tracker

UConn Facilities overtime equalization tracker. Tracks OT offers (offered, accepted, declined), hours worked by type (OT, vacation, sick, FMLA, comp, block, period), and equalization balance across crew members. Dark/light theme, responsive.

## Deploy

Push to `main` — GitHub Pages serves from root.
